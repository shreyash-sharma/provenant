# Benchmarks

This directory contains a compact, repo-local proof trail for the Provenant evaluation.

The full benchmark harness and paper artifacts are not published here yet, but the summary files capture the headline numbers used in the README and whitepaper.

## Files

- [`sample_results.json`](sample_results.json): machine-readable aggregate result summary.
- [`swebench_verified_summary.csv`](swebench_verified_summary.csv): CSV summary of SWE-bench Verified file-localization results.

## Scope

- Dataset: SWE-bench Verified
- Tasks: 500 real GitHub issues
- Repositories: 12 Python repositories
- Per-repo reporting: 9 repositories with at least 10 tasks
